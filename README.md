# naverworks-api-gateway

## Overview
- Naverworks API와 Jira & Confluence, Github, Figma 등 협업도구 연동을 위한 gateway

## Feature
- 토큰 관리
- Naverworks 직급 조회
- 협업도구 이벤트 발생 시 Bot message
- Bot 생성/수정 등 관리
- ChannelId 맵핑
